# 올인원장 도메인 컨텍스트

## 기업 소개

**올인원장(allinonezang)** 은 동네 소형·중형 학원의 원장을 위한 **학원 운영 통합 SaaS** 플랫폼입니다.

- **설립 배경**: 10년 이상 학원을 직접 운영한 현업 원장이 기획·개발에 참여 — "원장이 만든 원장용 도구"
- **핵심 미션**: 원장의 반복 행정 업무를 최소화하고, 절약한 시간을 수업·상담·홍보 등 매출 활동에 쓰게 하는 것
- **주요 성과**:
  - 기능 수 5개 → 39개로 확대 (약 680% 성장), 서버 인프라 12배 확장, 매출 3.5배 성장
  - 국내 최대 학원장 커뮤니티 '열피쌤'(회원 3만 2,000명) 기반 성장
  - 2026 한국 교육박람회, WIS 2026 참가
- **서비스 형태**: 웹 기반 (allinonezang.com)
- **핵심 성장 과제**: **신규 수강생 유치** — 원장이 외부 채널에서 학원을 잘 알릴수록 올인원장도 함께 성장

**핵심 가치**: 원장의 반복 업무 최소화 + 매출 전환율 향상

## 고객 세그먼트 (학원 원장)

| 세그먼트 | 특성 | 주요 홍보 니즈 |
|----------|------|----------------|
| 1인 운영 원장 | 30~50대, 강사 1~5명, 원생 30~150명 | 시간 없이 빠르게 쓸 홍보 초안 |
| 신규 개원 원장 | 막 학원을 연 예비·초기 원장 | 학원 인지도 확보, 신규 원생 유입 |
| 성장기 학원 | 원생 확대·분원 준비 단계 | 이벤트·프로모션으로 단기 가입 유도 |
| 학부모 (간접 타깃) | 콘텐츠 노출 대상 | 출석 알림·피드백으로 신뢰 형성 |

### 원장 페르소나 핵심
- **고민**: "홍보할 시간이 없다", "SNS는 어렵다", "경쟁 학원과 차별화가 안 된다"
- **의사결정 패턴**: 동료 원장(커뮤니티) 추천에 높은 신뢰 / 무료 체험·도입 후기 중시 / "기능"보다 "시간이 얼마나 절약되는가"에 관심
- **홍보 3대 목적**: ① 신기능 출시 → 기존 회원 재활성화 + 신규 유입 ② 이벤트/프로모션 → 단기 가입 유도 ③ 후기 → 신뢰도 구축

## 홍보 채널별 특성 (이 미션의 핵심)

### 블로그 (네이버 블로그 / 티스토리)
- **역할**: SEO 기반 장기 유입, 정보성·신뢰 채널
- **독자**: 학원 관리 솔루션을 검색하는 예비·신규 원장
- **권장 분량**: **500자 이상** (SEO 최소 기준), 소제목(H2) 2~3개로 구조화
- **톤**: 정보성·신뢰감, 전문적 어조, 소제목 활용. 도입부(공감) → 기능/혜택 설명 → 사례/수치 → CTA

### 인스타그램
- **역할**: 브랜드 인지도·감성 채널, 비주얼 중심
- **독자**: 30~40대 원장, 학부모, 교육 관심층
- **권장 분량**: **캡션 150자 이내 + 해시태그 5개**
- **톤**: 밝고 친근, 감성적, 공감 유도("이런 경험 있으시죠?"). 한 가지 포인트만 강조

### 카카오채널 (카카오톡 비즈니스)
- **역할**: 직접 행동 유도(CTA) 채널, 기존 팔로워 대상 즉시 소통
- **독자**: 이미 올인원장·해당 학원에 관심 있거나 가입한 사람
- **권장 분량**: **180자 이내**, 이모지 1~2개
- **톤**: 간결·명확, 행동 유도("바로 신청하기 / 지금 확인하기"), 긴박감 또는 친근함. 링크 없이 텍스트만 있으면 전환율 급락

## 왜 이 과제가 중요한가 (비즈니스 맥락)

올인원장의 고객인 원장 대부분은 **마케팅을 처음 하는 교육 실무자**입니다. 좋은 학원이라는 건 알지만 "어떻게 알려야 할지 모르는" 상태에 머뭅니다.

**3채널 홍보 초안 자동 생성이 비즈니스에 중요한 이유:**

1. **원장의 시간 = 올인원장의 가치**: 채널당 1~2시간씩 쓰던 글쓰기를 분 단위로 줄이면, 올인원장이 약속한 "반복 업무 최소화"가 홍보 영역까지 확장됨
2. **채널 특성 살린 홍보 = 신규 원생 유치 직결**: 인스타(인지) → 블로그(신뢰) → 카카오(전환) 퍼널이 작동해야 학원이 성장하고, 학원이 성장해야 올인원장도 성장
3. **3채널 복붙 방지**: 같은 글을 3채널에 그대로 올리면 채널 특성이 사라져 효과가 0에 가까움 — 채널별 차별화가 핵심 가치
4. **플랫폼 기능 확장의 씨앗**: 외부 홍보 초안 생성은 현재 플랫폼에 없는 기능 — 이 자동화가 검증되면 제품 기능으로 편입 가능
5. **type별 강조점 자동 판단**: 신기능·이벤트·후기는 강조해야 할 키워드와 톤이 다름 → 원장이 매번 고민하던 "무엇을 어디에 강조할지"를 대신 판단

## 참고 용어 사전

| 용어 | 정의 |
|------|------|
| **원장** | 학원을 운영하는 대표자. 올인원장의 핵심 고객 |
| **원생** | 학원에 등록된 학생 |
| **원비** | 학원 수업료 |
| **수납** | 원비·교재비 수금 및 결제 처리. 올인원장은 비대면 자동 청구 지원 |
| **보강** | 결석 학생을 위한 보충 수업. 올인원장은 보강 일정 자동 조정 |
| **출결 전광판** | 학원 입구 화면에 학생 이름·출석 여부를 실시간 표시하는 기능 |
| **학부모 알림** | 출결·공지를 학부모 카카오톡으로 자동 전송하는 기능 |
| **AI 피드백** | 출결·참여도·학습 패턴 데이터를 분석해 학생별로 자동 생성되는 학습 피드백 리포트 |
| **CTA** | Call To Action. 행동 유도 문구 (예: "지금 무료로 시작하기", "오늘 마감") |
| **SEO** | Search Engine Optimization. 검색 노출 최적화. 블로그 500자 이상 권장의 근거 |
| **퍼널(Funnel)** | 인지 → 관심 → 전환으로 이어지는 마케팅 단계. 인스타(인지)→블로그(신뢰)→카카오(전환) |
| **숏폼** | 릴스·쇼츠 등 10초~1분 단위 짧은 영상 콘텐츠. 학원 마케팅 비중 증가 추세 |
| **type(콘텐츠 유형)** | 이 미션의 3종 — 신기능 / 이벤트 / 후기. 유형마다 강조 키워드·톤·CTA가 다름 |
| **tone_hint** | 원장이 원하는 톤 힌트(전문적인/따뜻한/긴급한/자랑스러운/밝은/친근한 등). 채널 특성에 맞게 변형 적용 |
| **brand_name** | 케이스별 학원 이름. 각 채널 콘텐츠에 자연스럽게 노출 |
