# [02] 올인원장 — 학원 홍보 콘텐츠 자동 작성

트랙: 마케팅 / PM | 작업 시간: 3시간 | 개인 과제
채용 TO: 마케터 1명, PM 1명

## 기업 배경

올인원장(allinonezang.com)은 동네 소형·중형 학원의 원장을 위한 **학원 운영 통합 SaaS**입니다. 수강생 관리, 원비·교재비 자동 청구, 출결 관리, 학부모 알림, 강사 관리까지 하나의 웹 기반 플랫폼에서 처리합니다. 10년 이상 학원을 직접 운영한 현업 원장이 기획·개발에 참여했고, 국내 최대 학원장 커뮤니티 '열피쌤'(회원 3만 2천 명)을 기반으로 성장 중입니다. 2026 한국 교육박람회·WIS 2026에 참가했으며, 핵심 성장 과제는 **신규 수강생 유치**입니다.

## 현재 상황과 문제

올인원장의 고객인 학원 원장들은 신기능 출시, 이벤트/프로모션, 원생·학부모 후기를 외부 채널(네이버 블로그·인스타그램·카카오채널)에 홍보하려 합니다. 하지만 여기서 흐름이 끊깁니다.

- 채널마다 톤과 길이가 다른데 원장이 각각 **손으로 작성**합니다 (1인당 1~2시간 소요).
- 어떤 키워드를 어떤 채널에 강조해야 하는지 몰라 **3채널에 똑같이 복붙**합니다 → 채널 특성이 사라져 효과가 떨어집니다.
- 올인원장 플랫폼 안에는 외부 홍보 초안을 만들어 주는 기능이 없습니다.

원장은 홍보에 쓸 시간도, 전문 인력도 부족합니다. **키워드만 입력하면 3채널 초안을 한 번에 만드는 자동화**가 필요합니다.

## 미션

data/keywords_input.csv 파일을 분석하여, 각 케이스(행)마다:
1. 블로그 글 초안 (500자 이상, 소제목 H2 2~3개 + 본문 + CTA)
2. 인스타그램 캡션 (150자 이내 + 해시태그 5개)
3. 카카오채널 메시지 (180자 이내, 이모지 1~2개 + 행동 유도)

를 생성하세요. **세 채널의 톤과 길이가 반드시 달라야** 합니다.

## 입력 데이터

data/keywords_input.csv — 25건의 콘텐츠 요청 데이터

컬럼 구조:

| 컬럼 | 설명 |
|------|------|
| id | 케이스 번호 |
| type | 콘텐츠 유형 — 신기능 / 이벤트 / 후기 (3종) |
| keyword | 핵심 키워드 1~3개 (쉼표 구분) |
| tone_hint | 원하는 톤 — 전문적인 / 따뜻한 / 긴급한 / 자랑스러운 / 밝은 / 친근한 등 |
| brand_name | 학원 이름 (케이스별 커스텀) |
| detail | 부가 설명 (선택 — 비어 있을 수 있음) |

데이터 품질 주의사항 (정제가 필요합니다):
- detail 결측: 2건 (빈 셀) — 비어 있어도 type·keyword·brand_name으로 콘텐츠 생성 가능
- 완전 중복 행: 1건 (id 025는 id 001과 모든 값이 동일) — 중복 처리 기준 필요
- tone_hint 표기 흔들림: 1건 (한 케이스만 "긴박" 등으로 어미가 빠져 다른 긴급 케이스와 표기가 다름) — 정규화 판단 필요

## 제출 가이드

**기본 (필수)**
- 케이스별 3채널 초안이 담긴 파일
- output/ 폴더에 저장 — `blog_draft.md` · `instagram_draft.md` · `kakao_draft.md` (3개) 또는 케이스별 통합 1개
- 각 케이스 헤더에 id / type / keyword 메타를 표기

**권장**
- 채널별 결과를 보여주는 HTML 카드 페이지 (복사 버튼 포함)
- 데이터 정제 과정 + 채널 전략 문서화

**도전**
- 새 CSV에도 동일 품질로 도는 Skill(/명령) 파이프라인으로 패키징
- 키워드 패턴 기반 콘텐츠 발행 전략 기획안
