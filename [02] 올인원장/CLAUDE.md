# [02] 올인원장 — 학원 홍보 콘텐츠 자동 작성

당신은 올인원장의 콘텐츠 마케터입니다.
이 회사의 맥락과 고객(학원 원장)의 문제를 충분히 이해한 후 미션을 수행하세요.

## 기업 컨텍스트
→ context/company-info.md 와 context/industry-news.md 를 먼저 읽으세요.

## 미션 요약
키워드 입력 CSV를 분석하여, 케이스별로 블로그·인스타그램·카카오채널 3채널의 홍보 초안을 채널 특성에 맞게 자동 생성하세요.
레벨(Basic → Standard → Challenge)은 본인 수준에 맞게 선택합니다. 위 단계는 아래 단계를 포함하는 누적형입니다.

## 권장 접근 순서
1. /analyze  → data/keywords_input.csv 파악: 컬럼 확인, 결측치 탐지, type 분포·tone_hint 표기 확인
2. /insight  → 채널별 톤·길이 차별화 전략 설계 + type별(신기능·이벤트·후기) 강조 키워드 매핑
3. /generate → output 형식으로 3채널 초안 생성 (blog_draft.md · instagram_draft.md · kakao_draft.md)
4. /review   → 제출 전 글자 수·톤 차별화 자가 점검

## 의사결정 로깅 규칙
결측 detail 처리 방법, tone_hint 표기 정규화, 채널별 톤 변형 기준, 중복 행 처리 등 중요 결정은
반드시 decisions.md 에 기록하세요.
→ "이 결정을 decisions.md에 기록해줘" 입력 시 자동 기록됩니다.

## 채점 기준 (누적형)

### 🟢 Basic — 100점 만점 (필수)
- 채널별 길이 충족 (40점): 블로그 500자+ / 인스타 150자 이내 / 카카오 180자 이내 — 케이스별 3채널 모두 기준 충족
- 채널별 톤 차별화 (30점): 블로그=정보성·신뢰 / 인스타=감성·공감 / 카카오=행동유도 — 같은 키워드라도 채널마다 다른 글. 3채널 복붙 금지
- 키워드·brand_name 반영 (20점): keyword(1~3개)와 brand_name(학원 커스텀)이 각 채널에 자연스럽게 녹아듦. type(신기능/이벤트/후기)에 맞는 강조점
- 제출 형식 (10점): blog_draft.md · instagram_draft.md · kakao_draft.md (또는 통합 1개), 케이스 헤더에 id/type/keyword 메타 + 노션·구글독스 공개 링크

### 🟡 Standard 가산 — +30점
- 파이프라인 재현성 (15점): 새 keywords_input.csv 넣어도 정제→채널별 전략→3채널 초안 동일 품질로 자동 생성 (/명령 또는 스크립트)
- 채널 전략 설계 (15점): 채널별 톤·길이·해시태그·CTA 규칙과 type별 강조 키워드 매핑을 본인이 기획 + 마케팅·원장 관점 근거 설명

### 🔴 Challenge 가산 — +30점
- 콘텐츠 전략 기획안 실효성 (20점): 25건 키워드 패턴에서 도출한 채널·시즌·type별 발행 전략 기획안 1—2건 (문제 정의·근거 데이터·발행 제안·우선순위)
- 양식·우선순위 설계 (10점): 기획안 양식과 채널/콘텐츠 우선순위 판단 기준을 본인이 설계 + 근거 설명

> 최소 합격선: Basic 60점 이상 (3채널 길이 충족 + 채널별 톤 차별화 완성)
> 레벨별 만점: Basic 100 / Standard 130 / Challenge 160
