## 채널별 핵심 차별화 원칙

### 블로그 — "정보성·신뢰 채널"
- **독자**: 학원 관리 솔루션을 검색하는 예비/신규 원장
- **목적**: SEO 유입 + 장기 신뢰 구축
- **글자 수**: 반드시 500자 이상 (공백 포함)
- **구조**: 도입부(공감) → 소제목(H2) 2~3개로 기능/혜택 단락 → 사례·수치 → CTA
- **톤 적용 기준**: tone_hint를 유지하되 정보성 구조 안에서 표현
  - 전문적인: 수치·기능 스펙·비교 정보 중심
  - 따뜻한/친근한: 원장·학부모 경험담, 공감 언어
  - 밝은: 이벤트·시즌 기획 강조
  - 긴급한: 기간 한정·마감 임박
  - 자랑스러운: 성과·차별점·수치 부각

### 인스타그램 — "감성·공감 채널"
- **독자**: 30~40대 원장, 교육 관심 학부모
- **글자 수**: 캡션 본문 150자 이내 + 해시태그 5개
- **구조**: 공감 한 줄 → 핵심 메시지 → 행동 유도 한 줄 → 해시태그 5개
- **주의**: 정보가 많으면 안 읽음. 한 가지 포인트만 강조
- **해시태그 필수**: #올인원장 #학원관리 + 주제·brand 관련 3개로 총 5개

### 카카오채널 — "행동 유도 채널"
- **독자**: 이미 팔로워 등록된 원장·관심자 (관심도 높음)
- **글자 수**: 180자 이내 (이모지 포함)
- **구조**: 이모지 + 핵심 혜택 → 기간/조건 → CTA 링크
- **이모지**: 1~2개 (과도하면 스팸 느낌)
- **CTA 예시**: "지금 바로 확인하기", "무료 체험 신청", "오늘 마감"

## type별 채널 전략

### 신기능
| 채널 | 전략 |
|------|------|
| 블로그 | 기능 소개 + 사용법 + 도입 전후 비교 |
| 인스타 | "이런 불편함 있으셨죠?" 공감 + 신기능 한 줄 |
| 카카오 | "[신기능] 출시 알림 + 바로 써보기 링크" |

### 이벤트
| 채널 | 전략 |
|------|------|
| 블로그 | 이벤트 상세 + 참여 방법 + 혜택 정리 |
| 인스타 | 혜택 요약 + 기간 강조 + "저장해두세요" |
| 카카오 | 마감일·선착순 강조 + 신청 링크 |

### 후기
| 채널 | 전략 |
|------|------|
| 블로그 | 원장/학부모 인터뷰 형식, 수치 포함 |
| 인스타 | 후기 인용구 한 줄 + "나도 해봐야겠다" |
| 카카오 | 후기 핵심 한 줄 + 무료 체험 CTA |

## 데이터 예외 처리 전략

### detail 결측
- detail이 비어 있어도 type·keyword·brand_name으로 콘텐츠 생성 가능
- 키워드와 type을 바탕으로 자연스러운 맥락을 추론 → decisions.md에 기록

### 완전 중복 행
- 모든 값이 동일한 행은 1건만 생성하거나, 정책을 정해 처리 → decisions.md에 기록

### tone_hint 표기 흔들림
- 어미·표기만 다른 값은 표준 톤으로 정규화 (예: "긴박" → "긴급한") → decisions.md에 기록
